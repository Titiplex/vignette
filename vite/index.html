Vignettes for Endangered Language Preservation

Build visual scenarios, collect oral recordings, and support fairer multilingual data by documenting linguistic diversity with community-friendly tools.

1. Explore language data

Browse the language catalog and navigate language families to anchor your scenario work in rich linguistic context.

Open the language catalog

2. Create scenario vignettes

Create image-based scenario sequences for elicitation and narration tasks adapted to under-resourced and oral-first communities.

Create a scenario

3. Record oral responses

For each image, record and upload audio to preserve pronunciation, intonation and local expression patterns.

4. Reduce translation bias

By grounding data collection in local narratives rather than direct translation, Vignettes helps reduce translation-driven bias.
